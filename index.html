Angular Installation Procedure.

Open your terminal and follow the steps:

nvm 4.o

nvm install node

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash

nvm ls-remote

nvm install v14.1.0

ng --version

ng new project #press enter key

cd project

ng serve -o-

You will be ready to launch.
