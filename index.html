Angular Installation Procedure.

Open your terminal and follow the steps:

1.nvm 4.0

2.nvm install node

3.curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash

4.nvm ls-remote

5.nvm install v14.1.0

6.ng --version

7.ng new project #press enter key

8.cd project

9.ng serve -o-

You will be ready to launch the app.
